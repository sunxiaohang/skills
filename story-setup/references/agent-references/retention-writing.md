# Retention Writing — 签约向写作技法

网文平台的签约核心不是"写得好"，而是"留得住"。此文件提供将商业留存结构织入写作流程的具体技法。

## 一、读者反应追踪法

**原理**：读者读网文的心理状态不是"欣赏"，是"追问"。每一段文字读完，读者脑子里要么出现一个新问题，要么得到一个答案（然后立刻出现下一个问题）。没有任何问题出现的段落 = 留存断点。

**操作**：写完一段后暂停，问自己：

| 读者当前状态 | 判断 | 处理 |
|------------|------|------|
| "然后呢？""他怎么办？" | 最佳——急迫型追问 | 继续 |
| "这人是谁？""为什么？" | 好——好奇型追问 | 继续 |
| "会不会是……" | 及格——猜测型参与 | 给一点线索，别全给 |
| （无追问） | 危险——留存断点 | 回去改。插入冲突/悬念/反常细节 |

**密度要求**：连续 300 字无新追问 → 红线，必须打断。

---

## 二、爽点时机公式

**免费平台（七猫/番茄）的爽点时间窗**：

| 位置 | 要求 | 为什么 |
|------|------|--------|
| 0-50 字 | 冲突/动作/对话开场 | 读者在 App 列表页停留 3 秒决定要不要点进来 |
| 50-300 字 | 亮出类型标签 + 建立主角困境 | 读者在确认"这是我想看的那种文吗" |
| 300-800 字 | **第一个爽点/情绪高点** | 读者已经投入了 2 分钟，需要回报 |
| 800-1500 字 | 第二个情绪转折（虐或爽） | 维持阅读惯性 |
| 章末 | 强钩子——读者必须翻页 | 追读率的核心卡点 |

**爽点不是只有"打脸"。以下都是爽点**：
- 系统激活 / 金手指首次亮相
- 主角做出一个聪明/善良/狠辣的选择
- 一个被看不起的人露出真正的实力
- 一个高冷的人对主角露出破绽（"她算错账了"）
- 一个具体的小胜利（收到第一个铜板、多撑了一天）

## 三、情绪节拍设计

**原理**：网文读者的情绪耐受曲线很短。同一种情绪持续超过 2 分钟（约 800 字），大脑会自动降低敏感度——读者开始刷过去。

**节拍模板（2500 字章节）**：

```
0-200字:  冲突/压力（钩子）
200-500字: 压力加深或新信息（虐 or 疑）
500-800字: 转机/小胜利（爽）        ← 第一个爽点
800-1200字: 胜利的代价/新麻烦（虐）   ← 情绪回落
1200-1700字: 主角应对/主动选择（燃）  ← 第二个情绪高点
1700-2200字: 应对的初步效果 + 隐患（期待+不安）
2200-2500字: 章末钩子               ← 必须翻页
```

**检查方法**：写完一章后标注每一段的情绪标签（焦虑/好奇/温暖/燃/压抑/期待/恐惧），看单种情绪是否连续超过 3 段。

## 四、章末钩子执行

**禁止的收尾方式**（全部来自 AI 的写作惯性）：
- 氛围留白："天黑了。城外有人在死。他翻了个身。没睡。"
- 哲理升华："他终于明白了——在这个世界上……"
- 预告旁白："他不知道的是，更大的风暴即将来临。"
- 纯心理："他想，也许明天会不一样吧。"

**强钩子五型**（按效果排序）：

### 1. 倒计时（最强）
不是"下周"、"过几天"——必须是具体的、紧迫的。
- ✅ "还剩两天。"
- ✅ "卯时之前。"
- ✅ "天亮之前，苏老爷子不能死。"
- ❌ "下周商城刷新的时候……"（太远）

### 2. 直接威胁
一个即时的、具体的危险。
- ✅ 剑抵在喉咙上。门被推开。有人跟踪。
- ✅ "师姐让我告诉你——苏老爷子烧到说胡话了。他问你——'周一是什么时辰。'"

### 3. 承诺兑现
上一章埋的承诺，这一章末开始兑现——但只兑现一半。
- ✅ "你说的药——到了吗。" "到了。"
- ✅ 她攥紧药盒，转身的时候说了三个字。很轻。"别让我爹死。"

### 4. 身份揭示
一个名字、一个身份、一句暗示——让读者重新理解前文。
- ✅ 凶手的手上有仵作的茧。爷爷的手上也有。一模一样的位置。

### 5. 两难抉择
一个必须马上做的选择，怎么选都有代价。
- ⚠️ 慎用。每卷不超过 3 次，否则读者疲劳。

## 五、主角主动性的具体写法

**主动 ≠ 反抗**。拒绝别人 = 被动。在两个选项之间做出选择 = 主动。

**如何在 200 字内建立主角主动性**：
1. 给主角一个即时的、具体的、二选一的处境
2. 让主角选其中一个
3. 让这个选择透露主角的某个特质（善良/精明/狠/赌性）

**示例**：
- ❌ 被动：陈老板出八两买他的铺子。他不卖。（不卖 = 拒绝 = 被动）
- ✅ 主动：他只有一两碎银。打火机二两，盐三两。他选了打火机——赌这一周没人会拿盐来换他的命。（选 = 赌性 = 主动）

## 六、热词标签植入

**不要硬塞。** 标签应该通过剧情自然出现，而非旁白介绍。

| 标签 | 自然植入方式 |
|------|------------|
| 系统 | 界面浮现，商品列表，标价，余额不足 |
| 穿越 | 主角对比现代和古代的一个具体细节 |
| 重生 | 前世的一个具体记忆闪回 |
| 种田 | 具体的劳作动作——凿井、修渠、摆摊 |
| 饥荒/乱世 | 一个具体的饥饿/危险细节，不靠旁白说"那年大旱" |
| 打脸/逆袭 | 看不起主角的人看到结果时的具体反应 |

**密度标准**：七猫编辑公子川说开篇需要 7-10 个亮点（热词标签）。检查方法：数一下第一章里出现了多少个平台热门标签词的**具体场景**（不是"提到了"，是"演出来了"）。
