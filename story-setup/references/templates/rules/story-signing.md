# Story Signing Rules — 签约向写作规则

生成正文时自动生效。优先级高于 narrative-writer 默认行为。

## 一、读者反应法则（每 200 字检查一次）

**规则**：每写完约 200 字，问自己一句话——"读者脑子里现在有没有一个必须被回答的问题？"

这个问题必须是：
- "然后呢？他怎么办？"（急迫型——最佳）
- "这人是谁？她为什么这样？"（好奇型——次好）
- "会不会……？"（猜测型——及格）
- 不是："这里的环境描写挺有氛围的"（无问题 = 划走）

**连续 300 字没有新问题出现 → 立刻插入冲突/悬念/反常细节。**

## 二、情绪节拍律（每 500-800 字一个来回）

**规则**：情绪不能单方向走超过 800 字。

- 压抑超过 800 字 → 必须给一个小希望/小温暖/小幽默
- 爽超过 800 字 → 必须埋一个隐患/新压力/新威胁
- 两段情绪之间必须有**具体事件**驱动，不能"突然就开心了"

**有效的情绪来回**：「被催债→发现系统→买到打火机→被人当成怪物→有人来求火」

## 三、开篇 300 字铁律

**前 50 字**：一个正在发生的动作/对话/冲突。禁止静态描写/背景介绍/天气。
**前 150 字**：读者知道主角在什么处境里，而且处境里有一个即时的压力。
**前 300 字**：出现至少一个平台热门标签信号（系统/穿越/重生/种田/饥荒/乱世/商城/升级/逆袭/打脸）。

检查方法：把前 300 字单独拿出来读——不知道主角是谁、在干嘛、有什么麻烦 → 重写。

## 四、章末追读钩子（最高优先级）

**规则**：章末最后一句话之后，读者必须有一个"不点下一章不行"的理由。

**强制使用以下五型之一**（严禁留白/氛围/抒情收尾）：

| 类型 | 示例 | 效果强度 |
|------|------|:--:|
| 倒计时 | "还剩两天"/"卯时之前"/"天亮之前" | 最强 |
| 直接威胁 | 剑抵喉咙 / 被人盯上 / 门被推开 | 强 |
| 承诺兑现 | "你说的药——到了吗" / "我记住你了" | 强 |
| 身份揭示 | 一个名字 / 一个身份 / 一句暗示 | 中强 |
| 两难抉择 | "救她还是救你娘" | 强（但慎用，每卷不超过 3 次） |

**禁止**："他翻了个身。没睡。" "天黑了。城外有人在死。" → 这些是留白，不是钩子。

## 五、主角主动性底线

**规则**：第一章内，主角必须**主动做出至少 2 个选择**。不是"被迫接受"，是"在多个选项中选了一个"。

- 被动："陈老板要买铺子。他拒绝了。" ← 拒绝不是选择，是本能
- 主动："他只有一两碎银。打火机还是盐？他选了打火机。" ← 这是选择

**每章至少一次**：主角做了一件读者没想到但觉得合理的事。

## 六、出场人物管控

**规则**：第一章有名字+有台词+有描写的角色 ≤ 5 个。
配角超过这个数 → 合并角色，或推迟到第二章出场。

街上的路人、围观的群众 → 不算人数，但也不算"出场人物"。

## 七、对话的"潜台词"原则

**规则**：每段超过 3 轮的对话，至少有一个角色在说"不是他真正想说的"。

- 讨账的人在算你还能活几天 → 她想说的是"我不想逼死你"
- 当她的面再按一次打火机 → 他想说的是"我不是妖人，别怕"

说不出口的东西 > 说出口的东西。读者感觉到潜台词的时候，就会停下来品味——这是"感觉"的来源。

## 八、与 narrative-writer 既有规则的关系

以下 narrative-writer 规则**不受影响**，继续生效：
- 去AI味 6 Gate（禁用词、句式去套路、心理外化等）
- 三维度织入（发生/感知/反应）
- 格式规范（段落长度、对话独立、禁止空行）

以下规则以本文为准（本文优先）：
- 开篇创作 → 以"开篇 300 字铁律"为准
- 收尾创作 → 以"章末追读钩子"为准
- 情绪弧线 → 以"情绪节拍律"为准

## 九、生成后自检（必须执行）

每章写完，回答这三个问题（不通过 = 没有完成，重写到通过为止）：

1. 前 300 字能单独拿出来，让一个没读过这本书的人说出"主角是谁、在干嘛、有什么麻烦"吗？
2. 章末最后一句话之后，读者脑子里是否有一个**今晚就必须被回答**的问题？
3. 这一章里，主角主动做的选择 ≥ 2 个吗？

三项全过 → 完成。任何一项不过 → 定位问题段落，重写。
