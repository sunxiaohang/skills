# 🎮 《几何冲刺 · GeoDash》— 微信小游戏策划案

> **版本**: v1.0  
> **日期**: 2026-03-31  
> **类型**: 伪3D透视跑酷 · 微信小游戏  
> **开发周期**: 2-4 周  

---

## 一、游戏概述

### 1.1 一句话描述

> 在无尽的低多边形几何世界中高速奔跑，躲避障碍、收集能量，挑战你的反应极限。

### 1.2 核心卖点

| # | 卖点 | 说明 |
|---|------|------|
| 1 | **伪3D透视冲击感** | 利用 Canvas 2D 实现透视消失点效果，赛道向远方延伸，营造强烈的速度感 |
| 2 | **低多边形几何美学** | 简约而不简单，用纯色几何体构建视觉世界，渲染性能优异 |
| 3 | **一指操控** | 左右滑动切换赛道 + 点击跳跃，极低学习成本 |
| 4 | **节奏感设计** | 障碍物分布与背景音乐节拍挂钩，越跑越"上头" |
| 5 | **社交裂变** | 好友助力复活机制，自然带来分享传播 |

### 1.3 目标用户

- **核心用户**: 18-35岁年轻男性
- **用户场景**: 通勤路上、排队等待、课间休息等碎片化时间
- **用户心理**: 追求刺激感、挑战极限、攀比排名

## 二、核心玩法

### 2.1 游戏机制

```
┌─────────────────────────────────────────────┐
│               消失点 (远方)                    │
│                  ·                            │
│                / | \                          │
│              /   |   \                        │
│            /     |     \                      │
│          /  左赛道 | 右赛道 \                   │
│        /    中赛道  |        \                  │
│      ─────────────────────────                │
│              ▲ 玩家角色                        │
│         (屏幕底部 1/4 处)                      │
└─────────────────────────────────────────────┘
```

- **赛道**: 3条平行赛道（左/中/右），通过透视投影在屏幕上呈现梯形收缩效果
- **移动**: 左右滑动切换赛道（约 0.15s 过渡动画）
- **跳跃**: 点击屏幕跳跃，躲避地面障碍
- **下滑**: 下滑手势，躲避空中障碍
- **速度**: 随跑动距离逐步加速，每跑 500m 提速一档，共 8 档

### 2.2 障碍物系统

| 障碍物类型 | 外观描述 | 躲避方式 | 出现频率 |
|-----------|---------|---------|---------|
| **几何方块** | 低多边形立方体，带棱角光影 | 切换赛道 | ★★★★★ |
| **三角刺** | 地面尖锐三角锥 | 跳跃 | ★★★★ |
| **横梁** | 横跨赛道的几何梁 | 下滑 | ★★★ |
| **移动棱柱** | 在赛道间左右移动的障碍 | 观察节奏后切换 | ★★ |
| **碎裂地面** | 赛道地面出现裂缝并塌陷 | 提前切换赛道 | ★★ |
| **激光门** | 间歇性开关的光门 | 观察节奏通过 | ★ |

### 2.3 道具系统

| 道具 | 图标 | 效果 | 持续时间 |
|------|------|------|---------|
| **能量晶体** | 💎 菱形多面体 | 基础得分 +10 | 即时 |
| **磁铁** | 🧲 几何磁铁 | 自动吸附周围能量晶体 | 5秒 |
| **护盾** | 🛡️ 六边形护盾 | 免疫一次碰撞 | 单次 |
| **加速** | ⚡ 闪电棱柱 | 无敌 + 极速冲刺 | 3秒 |
| **x2得分** | ✨ 双星体 | 得分翻倍 | 8秒 |

### 2.4 难度曲线

```
难度
 ▲
 │                                    ┌── 地狱模式 (3000m+)
 │                              ┌─────┘   障碍密集+移动障碍+激光门
 │                        ┌─────┘
 │                  ┌─────┘         ← 困难 (1500m-3000m)
 │            ┌─────┘                 速度提升+复合障碍
 │      ┌─────┘
 │ ─────┘                           ← 简单 (0-500m)
 │  新手保护期                         低速+单一障碍+道具密集
 └──────────────────────────────────→ 跑动距离(m)
   0    500   1000  1500  2000  3000
```

## 三、美术设计

### 3.1 视觉风格 — Low Poly 几何风

**设计关键词**: 简约、干净、几何、光影对比、色彩分层

#### 配色方案 (Material Design 3 基础)

| 层级 | 颜色 | 色值 | 用途 |
|------|------|------|------|
| **背景天空** | 深空蓝 → 暮光紫渐变 | `#0D1B2A` → `#3A0CA3` | 背景天空，随距离变化 |
| **赛道表面** | 石墨灰 | `#2B2D42` | 赛道主体 |
| **赛道边线** | 电光青 | `#00F5D4` | 赛道边缘发光线 |
| **障碍物** | 警戒红 | `#EF233C` | 所有障碍物主色 |
| **能量晶体** | 琥珀金 | `#FFB703` | 可收集道具 |
| **玩家角色** | 荧光蓝 | `#4CC9F0` | 玩家几何体 |
| **UI 强调** | 活力紫 | `#7209B7` | 按钮、得分等 UI 元素 |

#### 赛道环境随距离切换

| 距离 | 环境主题 | 背景色调 | 特殊元素 |
|------|---------|---------|---------|
| 0-500m | 极简白境 | 浅灰-白 | 干净几何，柔和阴影 |
| 500-1500m | 霓虹都市 | 深蓝-紫 | 赛道两侧几何建筑剪影 |
| 1500-3000m | 熔岩深渊 | 暗红-橙 | 地面裂缝发光，粒子飘浮 |
| 3000m+ | 数据虚空 | 纯黑-青 | 矩阵代码雨效果，闪烁线条 |

### 3.2 玩家角色

- **默认形态**: 发光的低多边形正二十面体（icosahedron），带尾部粒子拖尾
- **跳跃动画**: 压缩 → 弹起 → 旋转 → 落地
- **碰撞动画**: 碎裂成多个三角面片，向四周飞散
- **皮肤系统**: 不同几何体形态（立方体、八面体、十二面体等），纯靠游戏内解锁

### 3.3 UI 设计规范

- **字体**: 使用系统无衬线字体，粗体用于分数
- **HUD 布局**:
  ```
  ┌────────────────────────────────┐
  │ 💎 12,580     🏃 1,247m        │  ← 顶部：得分 + 距离
  │                                │
  │                                │
  │        [ 游戏主画面 ]           │
  │                                │
  │                                │
  │              ⏸️                │  ← 右上角：暂停按钮
  └────────────────────────────────┘
  ```

## 四、关卡与进度系统

### 4.1 无尽模式（核心）

- 永无止境地奔跑，以距离和得分计算成绩
- 每次死亡记录最高分和最远距离
- 全局排行榜 + 好友排行榜

### 4.2 挑战模式（第二阶段）

| 挑战类型 | 描述 | 奖励 |
|---------|------|------|
| **每日挑战** | 每天一个固定种子的关卡，全服同一赛道 | 限定皮肤碎片 |
| **限时冲刺** | 60秒内跑最远距离 | 能量晶体 x3 |
| **零失误** | 不碰任何障碍跑完指定距离 | 稀有皮肤 |

### 4.3 成就系统

| 成就名 | 条件 | 奖励 |
|--------|------|------|
| 初次起步 | 第一次游戏 | 🔓 立方体皮肤 |
| 千米选手 | 单次跑过 1000m | 💎 x500 |
| 速度恶魔 | 达到最高速度档 | 🔓 闪电特效 |
| 不死传说 | 利用好友复活续命 3 次并跑过 2000m | 🔓 金色皮肤 |
| 社交达人 | 成功邀请 5 位好友 | 🔓 限定几何体 |

## 六、技术方案

### 6.1 技术栈

| 项目 | 方案 |
|------|------|
| **引擎** | 微信小游戏原生 Canvas 2D API |
| **渲染** | 自研伪3D透视渲染管线（基于 Canvas 2D） |
| **物理** | 轻量碰撞检测（AABB + 赛道索引） |
| **音频** | 微信 InnerAudioContext API |
| **社交** | 微信开放数据域（排行榜）+ 分享 API |
| **数据** | 本地 wx.setStorageSync + 云开发数据库 |

### 6.2 伪3D透视实现原理

```javascript
// 核心透视投影公式
function projectToScreen(worldX, worldZ, cameraZ) {
    const scale = focalLength / (worldZ - cameraZ);
    const screenX = centerX + worldX * scale;
    const screenY = horizonY + (groundY - horizonY) * scale;
    return { x: screenX, y: screenY, scale: scale };
}

// 赛道绘制：从远到近逐段绘制梯形
for (let z = farZ; z >= nearZ; z -= segmentDepth) {
    const p1 = projectToScreen(-roadWidth, z, cameraZ);
    const p2 = projectToScreen(+roadWidth, z, cameraZ);
    const p3 = projectToScreen(+roadWidth, z + segmentDepth, cameraZ);
    const p4 = projectToScreen(-roadWidth, z + segmentDepth, cameraZ);
    drawTrapezoid(ctx, p1, p2, p3, p4, getSegmentColor(z));
}
```

### 6.3 性能优化要点

| 优化项 | 策略 |
|--------|------|
| **对象池** | 障碍物、道具、粒子全部使用对象池复用 |
| **视锥裁剪** | 只渲染玩家前方可视范围内的对象 |
| **分层渲染** | 背景层（低频更新）+ 游戏层（60fps）+ UI层（事件驱动） |
| **纹理缓存** | 几何体预渲染到离屏 Canvas，运行时直接 drawImage |
| **帧率控制** | 目标 60fps，低端机自动降级到 30fps |

### 6.4 包体控制

| 内容 | 预估大小 |
|------|---------|
| 游戏代码（压缩后） | ~300KB |
| UI 资源 | ~200KB |
| 音效文件 | ~500KB |
| 字体 | 系统字体，0KB |
| **总计** | **~1MB** (远低于 4MB 限制) |

## 七、数据埋点

### 7.1 关键指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| **次留** | 次日留存率 | ≥ 25% |
| **7留** | 7日留存率 | ≥ 10% |
| **场均时长** | 单次游戏平均时长 | 45-90秒 |
| **日均局数** | 每用户每天平均游戏次数 | ≥ 5局 |
| **分享率** | 产生分享行为的用户占比 | ≥ 15% |
| **助力转化** | 收到助力链接后进入游戏的比率 | ≥ 30% |

### 7.2 埋点事件

```
game_start        — 游戏开始

share_score       — 分享战绩
rank_view         — 查看排行榜
scene_change      — 场景切换 (附带: 新场景名)
```

## 九、风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 伪3D效果在低端机卡顿 | 用户流失 | 分级渲染 + 自动降帧 + 减少粒子 |
| 社交裂变效果不达预期 | 用户增长慢 | A/B 测试不同复活奖励力度 |
| 玩法同质化 | 用户无新鲜感 | 多环境主题 + 每日挑战保持新鲜 |
| 包体超限 | 无法提审 | 音效动态下载 + 资源懒加载 |
| 审核不通过 | 延期上线 | 提前检查内容合规，避免敏感元素 |

## 十、总结

**《几何冲刺 · GeoDash》** 是一款面向年轻男性用户的伪3D透视跑酷微信小游戏，以低多边形几何美术风格为特色，通过极简操控 + 节奏感障碍设计 + 好友助力复活的社交裂变机制，打造"30秒上手、停不下来"的游戏体验。

**核心公式**:  
> 伪3D速度感 × 低多边形美学 × 好友社交裂变 = 自传播跑酷爆款
