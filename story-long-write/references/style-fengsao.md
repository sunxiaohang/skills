# 疯骚搞笑意文风指南

> 用于都市灵异、程序员职业志怪等需要叙述态度的题材。
> 在 Phase 4 正文写作前按需加载。

## 三层叠加定义

| 层 | 含义 | 写法 |
|------|------|------|
| **疯** | 叙述有攻击性 | 句子敢短敢碎。情绪敢大。比喻往意想不到的方向拐。写恐惧不收着。 |
| **骚** | 叙述有态度 | 不是一个中立的摄像机。是一个有脾气有幽默感的旁观者。可以吐槽角色，可以吐槽世界。 |
| **搞笑** | 黑色幽默优先 | 日常细节里藏着笑点。不是抖包袱——是一个精准的比喻让读者在恐惧中间笑出声。笑完继续害怕。 |

## 句式节奏

- 短句为主（15-22字/句），长句用来落地一个笑点或冲击
- 段落 ≤60 字，一句一段为主
- 对话和叙述交错，不让叙述连续超过5句

## 比喻规范

比喻必须往意想不到的方向拐。禁止万能比喻（潮水般、闪电般）。用日常生活中的东西做喻体：

```
✅ 「像一个外卖员把所有门牌号摘下来重新挂到别人家门口」
✅ 「像全世界同时穿错了裤子」
✅ 「头发和服务器容量是反向增长的」
✅ 「阿杰的字典里没有这两个字——今天他自己凿了两个出来」
❌ 「像潮水般涌来」
❌ 「如闪电般迅速」
```

## 技术名词转化

所有编程术语必须包裹在人话里。不写技术名词，只写它造成的结果：

```
❌ session错位
✅ 身份自己跑了

❌ 回滚失败  
✅ 撤销键按烂了也回不去

❌ 权限校验函数
✅ 决定一个人能看到什么的那段代码

❌ 生产环境自动部署
✅ 自己往线上跑，没人按过启动键
```

## 句式禁令（强化版）

### 一级禁令：不是A，是B（★★★★★ 最高毒级）

已在 banned-words.md 中定义。**叙述中零容忍。对话中例外。**

```
❌ 不是密码被盗，不是系统被黑，是身份自己跑了。
✅ 身份自己跑了。密码没被盗，系统也没被黑——就是身份自己收拾东西跑了。

❌ 她脸上的表情——不是害怕，不是紧张。是那种……
✅ 她脸上的表情——那种……

❌ 不是耍酷——是他想了太久。
✅ 他想这件事想了太久，再想就要把脑子想炸了。
```

### 二级禁令：万能状语（★★★★）
```
❌ 他笑了一下，带着一丝不易察觉的嘲讽
✅ 他笑了一下。没到眼底。
```

### 三级禁令：直接情绪陈述（★★★★）
```
❌ 他感到恐惧 / 他觉得不对劲
✅ 后背的汗毛从后颈一路竖到腰
```

## 情绪交付

每章必须交付一个明确情绪弧线，不能只推进剧情：
- 温暖 → 不安 → 恐惧 → 释放 → 新恐惧
- 或：压抑 → 爆发 → 余韵
- 章节结尾必须有读者翻页的冲动
