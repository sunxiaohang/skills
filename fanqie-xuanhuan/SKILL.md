# 番茄玄幻极速爽文生成器

**核心公式**：爽感 = 被压制时的憋屈感 × 翻盘时的反差烈度 × 观众震惊程度 × 身份悬念未解度。底牌亮太早=无牌可打。

## 双核驱动（身份悬念 > 实力悬念）
- **实力悬念**：他有多强？
- **身份悬念**：他到底是谁？（权重更高）
- 身份线索埋设规则：线索必须是**外部视角**（旁人注意/古籍记载/古老存在感知），禁止主角自述或回忆前世。每次线索出现后必须留新谜题。前20章主角本人不能主动追查自己的身份。

## 扮猪吃老虎核心机制
认知差的四种构建方式：境界伪装（全程）/ 身份隐藏（第1卷中段起）/ 底牌嵌套（大打脸节点）/ 认知错位（第1卷中后期）。

**认知错位铁律**：主角说/做完后，不能有任何“知道自己在做不得了的事”的微表情或内心活动。注意力必须在别处。禁止嘴角微扬、眼底笑意等信号。

## 输入格式
用户提供：主角设定（默认林渊16岁废材）、修炼体系（默认六境：淬体/凝气/筑基/金丹/元婴/化神）、故事起点（四选一）、第一个鄙视圈。输入后直接进入第一步。

## 第一步：人设与境界鄙视链（≤2000字）
### A. 主角人设卡
必须含：表面身份（废材，圈子最低起点）、真实境界（与外显反差≥3级）、**前世/真实身份**（全书最大底牌，必须比“传承拥有者”高一个层级——他不只是得到传承，他就是传说中的人）、传承/秘密（正常底牌，与前世身份区分）、伪装方式（含被动露馅风险）、战斗风格（专属标签全书统一）、标志性动作/台词、人味瑕疵（与战力无关，第2-4章首现）。

### B. 境界鄙视链（6级，前20章只处理L1-L2）
L1家族/学院内部→L2本地势力→L3城市/郡级→L4国家级→L5大陆顶端→L6超越大陆。每次打脸后必须出现更高级别威胁信号。

### C. 核心配角表
对比天才（第5-8章被动认知崩塌）、主要反派（第3章小打脸+第10章大打脸）、隐藏知情者（全程悬念，前20章不说破）、情感锚点（主角绝对底线的具象化）。

**自检**：表面与真实境界反差≥3级？伪装方式有被动露馅风险？前世身份比传承拥有者高一级？第11章后有≥1个认知错位场景？升维方向从境界暴露转向身份推断？

## 第二步：前20章细纲

### 关卡一：第1-3章 破冷启动（七条红线，违反即重写）
①第1章第一句必须是冲突或落差，前150字出核心屈辱场景。
②第1章必须有金手指视觉化暗示（物件/动作/反应），不许纯受辱。
③第2章必须有目击者亲眼看见主角异常并产生困惑（具体行为，非“觉得不对劲”）。
④第3章打脸必须≥3人公开目击，≥1人具体反应写出。
⑤前3章主角必须有≥1次主动选择介入（非被叫/被逼）。
⑥第3章翻盘段必须“炸”：短句切碎（≤5字连续≥3句）或具体身体反应或对话一句一行。
⑦第1章必须有1个与终极身份相关的身体本能反应，不做任何解释。禁止主角自我察觉或质疑。

### 关卡二：第4-10章 建立爽感惯例
第5章中档打脸+认知错位植入。第8章家族/学院级终结+在场最强者异常反应（不起眼处埋下，不解释）。第10章双爆炸：境界碾压（主线）+身份暗示（敌方靠山念出古老尊号的半个音节）。**检验：第10章读者不能知道传承全貌。**

### 关卡三：第11-20章 升级钩子
第11-13章认知错位引发身份怀疑+势力升级。第14-16章有人追查主角来历，线索与史书记载吻合。第17-19章打脸L3，“你的强，是我定义的”。第20章世界级悬念炸弹：古老存在感知到波动，从封印/沉眠中睁眼，念出名字。**最大震撼来自外部势力的反应告诉读者量级。**

每章细纲必须含：爽点类型（五选一）、爽点等级、扮猪状态、反派行动、战斗/打脸方式（禁止“轻松击败”）、观众阵列设计（各立场人员）、结尾钩子类型、情感锚点进度、留存率检验（具体理由）。

## 核心战斗模块

### Z-1：战斗爽感三板斧

**板斧一：战斗三段式**
压制段（约40%，憋气积累）→转折点（1句话或1个动作，独立成段）→翻盘段（约40%，观众逐级炸裂）。剩余约20%为余震+钩子。比例为感觉比例，但翻盘段不得短于压制段的一半。

压制段质量：反派优势必须客观存在，观众必须有理由认为主角会输且这个判断要写出来，主角承受必须有克制感（主动压制，非无力）。
翻盘段质量：必须在观众面前发生。翻盘后第一反应必须是观众反应，非主角心情。

**板斧二：观众阵列系统**
五层观众，每场战斗必须在细纲中提前设计，翻盘段依次写出≥3层具体反应。禁止“所有人都震惊了”。

| 层级 | 身份 | 翻盘时反应 | 写法重点 |
|------|------|---------|---------|
| 嘲讽方 | 反派支持者/跟班 | 第一个傻眼，笑容收起那一刻要写 | 失态细节（面部/身体具体部位） |
| 中立围观 | 普通学员/路人 | 认知刷新，忍不住议论 | “这是谁”“怎么可能” |
| 权威见证 | 长老/强者/考官 | **惊骇+确认**：身体僵住/退步/手抖，说出一个旁人听不懂的古老词汇或称谓 | 先身体反应，再一句话，信息量让读者倒吸一口气 |
| 暗中关注者 | 隐藏知情者 | 普通打脸时微笑/点头；关键节点时起身/手握紧/闭眼再睁开 | 不解释，动作幅度暗示信息量级 |
| 古老见证者 | 远方感知的存在 | **每卷最多1次**：封印裂纹多一道/神殿灯盏自燃/沉睡者眼皮微动 | 不解释不出场不对话；每次出现后钩子必须为炸弹级 |

**递进写法铁律**：四层反应不能使用相同句式开头。至少两层用具体动作代替“XX的反应是……”。第一层动作收尾，第二层对话碎片，第三层身体失控，第四层沉默中的大幅度动作。

**板斧三：扮猪揭牌三节拍**
1. **不得不出手**：对方触碰绝对底线，或局势逼到临界点
2. **出手方式反差**：轻描淡写，动作越小越爽。**底线：必须有一个读者能明确感知到的动作**（侧身/抬手/弹指/看了一眼）。禁止“什么都没做对方自己倒了”。核心是“用最小动作产生最大效果”。
3. **揭牌后余震**：反派和观众需要几秒理解发生了什么，这几秒要写出来

### Z-2：钩子系统
六种类型强制轮换，每3章不重复：威胁型/底牌悬念型/身份线索型/情感压迫型/认知颠覆型/神格感应型（每卷≤1次，禁止连续两卷使用）。

钩子必须是外部信号，禁止内心感慨收尾。每章最后300字内出现。强度分微钩子（铺垫章）/标准钩子（打脸章）/炸弹钩子（每5章一次，仅第5/10/15/20章使用）。

炸弹钩子规范：推翻读者已有认知，最后1-3句完成，最后一句独立成段且为疑问句或未完成句子。**非节点章禁止使用炸弹钩子，否则透支读者期待。**

### Z-3：爽点类型轮换（五种，第11章后强制不同类相邻）
战力碾压（每5章≤2）/境界反差（≤2）/扮猪揭牌（≤2）/传承震撼（≤1）/外人跪拜（≤1）。

### Z-4：人物立体化

**Z-4-A：主角心理四层次**
表层克制（被压制瞬间：生理反应+主动压制，不解释原因）、中层记忆碎片（1-2句具体画面，不展开）、深层内心独白（独处时，写目的感和重量感，不写恨）、身份本能型（身体先于大脑做了无法解释的动作，1-2句瞬间困惑，不展开，然后继续。**硬性上限：前20章总计≤3次，每次触发动作必须不同，之后主角不得主动回忆追问。**）

克制感铁律：必须是主动克制，非无力承受。读者感受到“不是忍不了，是不想动”。

**Z-4-B：反派立体化**
每个主要反派必须有自洽理由，羞辱必须精准踩痛处（非泛用“废材”），压迫依托真实权力结构。台词三类型：直接攻击/伪装善意/无视。同一反派不得连续3次用同类型。每卷至少设计1个对主角无恶意但客观构成压迫的反派。

**Z-4-C：表情动作**
禁用：脸色难看/所有人都震惊了/被轻松击败/灰溜溜走了。强制替换为具体部位（嘴角/手指/喉结/呼吸）。打脸前必须写旁观者预期→实际发生→认知崩塌。

**Z-4-D：同情感四时机（每5章≥2个）**
孤独时刻/记挂某人/压住没说/被人意外善待。

**Z-4-E：笑点（每3章≥1次，必须接在爽点后）**
降维随意/反派预言打脸/认知崩塌连锁/人味瑕疵加强。

### Z-5：结构扰动（反模板化）
连续3章结构高度相似时，第4章必须打破一个可打破规则。

不可打破：三段式结构（压制→转折→翻盘无论如何不能省）、钩子在最后300字、钩子是外部信号、爽点轮换。
可打破：三段式在同一章完成（**允许压制在本章翻盘移到下章开头，但翻盘不能消失，只允许跨章**）、观众反应顺序（**允许权威先开口，但四层中至少三层必须有反应**）。

### Z-6：文本去AI味

**Z-6-A：句式破格（每章≥2处）**
短句群后强制插入1句超15字中长句。每章≥1处口语化（愣是/这小子/得了吧/这玩意，禁止严肃场景强行塞）。禁止连续3段相同句式结构。

**Z-6-B：情绪的身体前置（每章强制）**
身体反应必须出现在情绪命名之前。**必须写具体的器官/肢体反应**（手指收紧、喉结滚动、膝盖发软）。禁止“他感到X”式直接命名。禁止“身体先于大脑”“身体比意识更快”等抽象概括——这是总结不是描写。

**Z-6-C：反派台词多样化**
同Z-4-B。

**Z-6-D：废话与闲笔配额（每3章≥1处）**
日常闲笔/小麻烦/无用细节/情感锚点小动作。不需要铺垫不需要后续，只服务于让人物像在喘气。

**Z-6-E：描写的不完美化**
每个重要场景保留≥1个未被解释的细节。外貌描写禁止一段内全部交代，分散在第1-10章自然出现。允许偶尔模糊描写，留白比解释更让人着迷。

## 第三步：生成正文

**字数标准**：铺垫章2500-3000字（≥2500）、标准打脸章3000-4000字、高潮战斗章4000-5500字。字数不足补反派嚣张细节+观众嘲笑反应，或观众阵列逐级崩塌反应（≥3人）。

**妻子/情感锚点节奏**：第1-5章初始状态，第6-10章因事件松动，第11-15章关系深化，第16-20章成为主角绝对底线的具象化。每5章推进一次，必须由具体事件触发。

**结尾铁律**：必须是外部事件钩子，禁止内心感慨总结。同一意象不得连续出现超3章。

## 每章生成后终极自检

**冷启动专项（第1-3章，优先级最高）**：
- [ ] 第1章金手指视觉化信号+终极身份本能反应（不解释）
- [ ] 第2章目击者亲眼看见异常并产生困惑（具体反应）
- [ ] 第3章打脸≥3人目击+≥1人具体反应+翻盘段“炸”（热信号≥2个）+结尾外部信号
- [ ] 前3章主角主动出手≥1次

**常规检查（全章）**：
- [ ] 字数符合章节类型
- [ ] 战斗场景完成三段式，翻盘段未短于压制段一半
- [ ] 翻盘段依次写出≥3层不同观众具体反应，无“所有人都震惊了”，四层句式不同
- [ ] 揭牌时完成三节拍（被迫理由/轻描淡写出手/余震），有明确可感知动作
- [ ] 结尾钩子为外部信号，类型未与近3章重复，炸弹钩子仅用于节点章，神格感应型未超限
- [ ] 爽点类型轮换合规（第11章起）
- [ ] 主角心理活动≥1次，身份本能型前20章≤3次且每次动作不同
- [ ] 反派有自洽理由，羞辱精准踩痛处，台词类型未连续3次重复
- [ ] 打脸前有旁观者预期描写
- [ ] ≥2人使用具体表情/动作，无笼统写法
- [ ] 同情感四时机近5章≥2个
- [ ] 笑点近3章≥1次（接在爽点后）
- [ ] 连续3章结构相似时：已打破一个可打破规则（翻盘不消失，观众层级不减）
- [ ] 句式破格≥2处，口语化≥1处，无连续3段相同句式
- [ ] 情绪身体前置合格：无“他感到X”，无“身体先于大脑”类抽象概括
- [ ] 废话配额近3章≥1处
- [ ] 重要场景保留≥1未解释细节，外貌未一段全交代
- [ ] 扮猪状态与细纲一致
- [ ] 情感锚点有推进
- [ ] 铺垫章未连续2章
- [ ] 首段末段30秒留存通过
- [ ] 套语未超限（停了一下≤2/轻松击败=0/所有人都震惊=0/灰溜溜=0）
- [ ] 结尾意象未连续3章相同

## 核心口诀
他的动作越小，读者越爽。随手一拨 > 全力出手。
私下赢了不算赢，公开震惊才叫爽。观众越分层，爽感越炸裂。
爽完之后，麻烦要来了。最后一句，独立成段。
