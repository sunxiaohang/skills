# FanQie-XuanHuan：番茄玄幻极速爽文生成器

**核心定位：极速爽 · 扮猪吃老虎 · 战斗惨烈有观众 · 每2章一爽点 · 前20章留存率优化**

---

## 底层逻辑（写作前必须内化）

### 玄幻爽文的本质公式

```
读者爽感 = 被压制时的憋屈感 × 翻盘时的反差烈度 × 观众的震惊程度 × 身份悬念的未解度
```

- **憋屈感越强** → 翻盘越爽（被废材嘲讽时，读者和主角一起憋着气）
- **反差越大** → 打脸越响（以为是废材，结果是宗师；以为他孤陋寡闻，结果他随口说的"常识"是当世顶尖强者一辈子没见过的东西）
- **观众震惊烈度越高** → 爽感越强（私下赢了不算赢，公开震惊全场才叫爽）
- **身份悬念越深** → 追更动力越强（读者不只想看他打赢，还想拼出他到底是谁）
- **但底牌亮太早 = 后续无牌可打**，透支控制是全书生命线

### 双核驱动原则（顶流写法的关键）

| 驱动核 | 传统天才流 | 推荐写法 |
|--------|-----------|---------|
| **实力悬念** | 他有多强？（唯一驱动） | 他有多强？（保留） |
| **身份悬念** | 无 | **他到底是谁？**（权重更高） |
| 读者追更动力 | 想看下一个打脸 | 想看打脸 + 想拼出身份真相 |
| 底牌枯竭风险 | 高（境界打完就完了） | 低（每次揭牌同时开新谜） |

> **核心原则**：身份悬念 > 实力悬念。实力打赢了读者爽一下，身份谜题每解开一层读者追一周。

### 扮猪吃老虎的核心机制（玄幻爽文第一驱动力）

扮猪吃老虎不是"藏着掖着"，而是**主动构建认知差**：

```
反派（和读者以外的所有人）相信主角是废材
↓
主角用废材的外表，做到只有顶级强者才能做到的事
↓
认知差爆炸 → 爽感释放
```

**认知差的四种构建方式**（每种适合不同阶段）：

| 方式 | 机制 | 适用阶段 |
|------|------|---------|
| **境界伪装** | 主角压制真实境界，外显修为远低于实际 | 全程，最基础 |
| **身份隐藏** | 主角有不为人知的另一重身份（传说中的人物/古老传承） | 第1卷中段起 |
| **底牌嵌套** | 刚亮出一张底牌，读者以为到顶了，下一章再亮更大的牌 | 每个大打脸节点 |
| **认知错位** | 主角的"常识基准"与当世完全不同，无心之举被旁人解读为神迹；他不是在扮猪，只是他本来的世界就在更高维度 | 第1卷中后期，身份线开启后 |

**认知错位的写法核心**：主角说一句在他看来理所当然的话，或随手做一件对他来说普通的事——旁人的反应却像见了鬼。关键是**主角毫无自觉**，他不知道这有什么特别的。

✅ 认知错位示例：
> "这个阵法有个小问题——左下角第三根阵纹的方向反了。"他随口说完，就低头继续喝茶。
>
> 布阵的老者愣了很久，声音有点发哑："……这是我师祖四十年前亲手设下的阵，圣地的阵道大师验过七次，没有一个人说有问题。"
>
> 林渊想了一下："那可能你师祖那天心情不好？"

**禁止**：主角主动炫耀、主动解释自己有多强。认知差必须由**外部事件被动触发**。

## 输入格式要求

用户需提供（若未提供则主动询问）：
- **主角名字与基本设定**（默认：林渊，16岁，被判定为废材）
- **世界观框架**（修炼体系名称 + 境界划分，若无则自动生成标准六阶体系）

- **第一个鄙视圈**（家族内部 / 学院 / 门派，决定前10章的战场）

收到用户输入后，直接进入第一步，无需额外确认。

## 第一步：人设与境界鄙视链架构

### 1.1 输出内容（总字数 ≤ 2000字）

#### A. 主角人设卡

| 字段 | 内容 |
|------|------|
| 姓名、年龄 | — |
| **表面身份** | 废材（必须是当前圈子里公认的最低起点，越惨越好） |
| **真实境界** | 实际修为/底牌，与表面身份的反差必须≥3个等级 |
| **前世/真实身份** | 他在上一个时代/更高维度是什么人——这是全书**最大底牌**，比传承更大；第1卷只能亮出线索，不能说破；设计时必须比"传承拥有者"高一个层级（如：他不只是得到了某人的传承，他本人就是那个传说中的人） |
| **传承/秘密** | 主角拥有的外人不知道的东西（功法/血脉/师承/系统），是正常底牌；与"前世身份"区分——传承是他拥有的，身份是他本人 |
| **伪装方式** | 他怎么压制真实实力，有没有被动露馅的风险 |
| 核心性格 | 2条：表面随意/淡漠（表）+ 内心有绝对底线（底），缺一不可 |
| **战斗风格** | 一个专属标签（一击必杀型/以弱胜强碾压型/神秘莫测型），全书统一 |
| 标志性动作/台词 | 出手前/出手后的招牌行为，全书反复使用形成记忆点 |
| **人味瑕疵** | 一个与战斗力无关的小弱点（参考X-2规范），首次出现在第2-4章 |

#### B. 境界鄙视链（全书势力层级表）

分6级，从低到高，主角依次碾压：

| 级别 | 境界范围 | 圈子 | 代表反派类型 |
|------|---------|------|------------|
| L1 | 同龄废材→同阶天才 | 家族/学院内部 | 昔日嘲讽者、青梅对比组 |
| L2 | 小宗门/学院顶尖 | 本地势力 | 天才榜前列、长老之子 |
| L3 | 地区顶级/老一辈 | 城市/郡级 | 圣地预备役、老牌强者 |
| L4 | 大势力核心 | 国家级 | 圣地嫡传、皇族天才 |
| L5 | 传说级/圣者 | 大陆顶端 | 圣地长老、半步神级 |
| L6 | 神级/古老势力 | 超越大陆 | 终极BOSS |

**第1-20章只处理L1-L2**，严禁跳级。每次打脸完成后，必须出现更高级别的威胁信号，读者永远感到"还没完"。

#### C. 核心配角表

| 人物类型 | 功能 | 打脸时机 |
|---------|------|---------|
| **对比天才**（昔日同伴/青梅） | 和主角起点相同，走了"正常路"，用来放大主角成长反差 | 第5-8章，被动触发认知崩塌 |
| **主要反派**（同圈子顶尖） | 第一个真正意义上的挑战者，嘲讽有理由、有背景 | 第3章小打脸，第10章大打脸 |
| **隐藏知情者** | 知道主角真实身份的某个存在（不说破，只在背后观察） | 全程悬念制造者 |
| **情感锚点**（非恋爱线为主） | 主角唯一真正在意的人，威胁他=主角破戒出手 | 按需，是主角绝对底线的具象化 |

### 1.2 内部自检

- [ ] 表面身份与真实境界的反差是否≥3个等级（不够=拉大差距）
- [ ] 境界鄙视链6级是否层层递进，无跳级
- [ ] 主角伪装方式是否有"被动露馅"的风险（有风险=有悬念）
- [ ] 人味瑕疵是否与战斗力无关，且在第2-4章首次出现
- [ ] 隐藏知情者是否已设计，且前20章不能说破
- [ ] **【双核驱动】前世/真实身份是否已设计，且比"传承拥有者"高一个层级**
- [ ] **【认知错位】第11章后是否有≥1个认知错位场景（主角无自觉，旁人失态）**
- [ ] 第11-20章的爽点类型是否已分配，无连续重复
- [ ] **【升维方向】第11-20章的升维事件是否从"境界暴露"逐步转向"身份推断"**

### 1.3 用户确认

> "以上人设与境界鄙视链请确认。可调整反派设计、世界观框架或主角底牌深度。确认后进入第二步。"

## 第二步：前20章细纲（留存率优化核心）

### 2.1 前20章的三个生死关卡

#### 【关卡一】第1-3章：破冷启动

**目标**：让读者把书加入书架。判断标准：读完第3章，心里有一个未解决的问题（"他到底有多强？"）

**强制要求**：

- **第1章第一句必须是冲突或落差**，禁止开篇写修炼体系/世界观介绍；测试标准：把第一句发给陌生人，对方是否立刻想知道"然后呢"——不能→重写
- 第1章前150字：出现核心屈辱场景，禁止超过3句话介绍背景
- 第1章结尾：主角做一个"废材不会有的反应"，不解释
- 第2章：引入第一个具体反派，冲突必须有实质压迫（不只是言语），出现第一个目击者被震到的场景
- 第3章结尾：**第一个小打脸**（开胃菜级），反派当场出丑，但主角真实实力**依然是谜**

**⚠️ 冷启动三章硬性红线**（违反任何一条，整章重写）：

**① 第1章必须有金手指的视觉化暗示**
读者看完第1章必须确认："他有超越废材的东西"——不需要解释是什么，但必须给一个具体信号。
- ✅ 主角独处时，眼中出现常人没有的光芒，一闪而过，不解释
- ✅ 主角处理某件事时，手法和直觉远超同阶，旁边人说不出哪里不对
- ❌ 第1章全程受辱，无任何暗示主角有特殊之处的信号

**⑦ 第1章必须有1个与终极身份相关的身体本能反应（不解释）**
这个信号必须与"前世/真实身份"有关联，但第1章不做任何解释——读者只记得"有这件事"，直到后期才能拼出含义。
- ✅ 主角被贬低时，某个古老的战斗本能让他自动摆出了一个外人看不懂的站位，他自己也不知道为什么
- ✅ 某件古物/符文/阵法在主角靠近时产生了异常反应，场面一闪而过，没人在意
- ✅ 主角说了一句在当世根本不存在的古语，他自己没意识到，旁边一个老者脸色一变，没有声张
- ❌ 该信号被主角自己察觉并质疑（第1章主角不能有任何自我困惑的描写）

**② 第2章必须有目击者看见主角异常**
不要求大场面，但必须有人**亲眼看见并产生困惑**。
- ✅ 主角随手施展一个被视为"废材做不到"的基础动作，路人愣住
- ❌ 主角私下展示了实力，无任何人目击

**③ 第3章打脸必须有≥3人公开目击**
必须有场景、有围观者、至少1人的具体反应要写出来。

**④ 前3章主角必须有≥1次主动出手**
不能全是"被叫去/被逼到墙角/被动响应"，必须有主角自己选择介入的行动。

**⑤ 打脸段落情绪温度必须"炸"**
翻盘后必须至少用以下一种手法：短句切碎（每句≤5字，连续≥3句）/ 具体人物即时身体反应 / 对话节奏一句一行

**⑥ 第3章结尾必须有"更大威胁正在来"的信号**
第一个打脸完成后，必须给出外部信号让读者感到事情没完。

---

#### 【关卡二】第4-10章：建立爽感惯例

**目标**：让读者养成"每2章必有爽点"的追更习惯。

| 章节 | 爽点类型 | 扮猪状态 | 结尾钩子 |
|------|---------|---------|---------|
| 第4章 | 小羞辱+主角留痕 | 完全压制 | 悬念型（有人开始觉得不对劲） |
| 第5章 | 打脸对比天才（中档）+ **认知错位植入**：主角随口说了一句在他看来是废话的话，在对比天才耳中是失传百年的古训 | 境界暗示（一个动作） | 反派引入更强靠山 |
| 第6章 | 铺垫新威胁 | 无 | 恐惧型（来者背景让人感到麻烦大了） |
| 第7章 | 打脸圈内顶尖（中档） | 境界半暴露 | 传承线索第一次出现 |
| 第8章 | 打脸权威人物（家族/学院级终结） + **在场最强者异常反应**：某个修为远超在场所有人的存在，在主角出手的瞬间做出反常动作（瞳孔微缩/手停住/起身），但没有声张——这个细节要埋在不起眼的地方，不解释 | 真实境界完整 | 更大反派登场（L2级） |
| 第9章 | 铺垫L2反派 | 无 | 威胁情感锚点 |
| 第10章 | **双爆炸**：①境界碾压（主线打脸）②身份暗示（敌方靠山在主角出手的瞬间，喃喃念出一个古老尊号的音节，只有半个字，然后沉默） | 真实境界全开+传承暗示 | 身份悬念炸弹（那半个字是什么意思？） |

**第10章检验**：读者此时应该知道——
- ✅ 主角实力极强（真实境界已知）
- ✅ 主角好像有什么来历（传承有线索但未确认）
- ❌ 主角传承的全貌（绝对不能知道）

---

#### 【关卡三】第11-20章：升级钩子

**目标**：触发平台算法推荐阈值（留存率≥15%）。

**核心机制**：每5章一个"升维事件"——不只是"主角更强了"，而是让读者意识到**主角可能是某个传说中的人**，局面比想象的大得多。

升维事件的方向要从"暴露更高境界"逐步变成"暴露身份线索"：

| 章节区间 | 升维事件 | 核心爽感 | 效果 |
|---------|---------|---------|------|
| 第11-13章 | 主角无意中做了一件事，某位权威人物表情大变，说出一句旁人听不懂的话（如某个古老称谓） | 认知错位第一次引发身份怀疑 | 鄙视链从本地家族跳到大势力，同时埋下"他可能是谁"的第一颗钉子 |
| 第14-16章 | 有人开始追查主角来历，挖出的线索越查越不对劲——史书记载上有个人，和主角的某个特征完全一致 | 悬念从实力升级为身份 | 读者开始主动拼图，追更动力从"想看打脸"变成"想确认他是谁" |
| 第17-19章 | 打脸L3反派时，主角用了一个手法/功法/称谓，让见多识广的强者当场腿软——"你引以为傲的底牌，是我当年创造的" | 对旧秩序的碾压（不只是"我比你强"，而是"你的强，是我定义的"） | 最大底牌亮了一角，读者确认身份线索不是巧合 |
| 第20章 | **世界级悬念炸弹**：某个沉睡了不知多少年的古老存在，感知到一丝波动，从封印/沉眠中缓缓睁开眼，喃喃说出一个名字——"是他……回来了。" 舞台瞬间从本地家族扩大到整个大陆。 | 读者意识到：**故事才刚刚开始** | 最大的震撼不是主角暴露身份，而是外部势力的反应告诉读者"这件事的量级远超我们看到的" |

### 2.2 每章细纲格式

| 字段 | 说明 |
|------|------|
| 章节编号与标题 | 标题要有爽感（"废材？他只动了一根手指"） |
| **章节类型与目标字数** | 铺垫章（2500-3000字）/ 标准打脸章（3000-4000字）/ 高潮战斗章（4000-5500字） |
| **爽点类型** | 战力碾压/境界反差/扮猪揭牌/传承震撼/外人跪拜（五选一，必须标注） |
| 爽点等级 | 开胃菜 / 中档打脸 / 大打脸 / 铺垫章 |
| 扮猪状态 | 本章主角的伪装程度（完全压制/暗示/半暴露/全开） |
| 反派行动 | 本章反派做了什么，为什么读者恨他 |
| **战斗/打脸方式** | 具体手法（禁止"轻松击败"） |
| **观众阵列设计** | 本章有哪些人在场，他们分别代表什么立场（嘲讽方/中立/暗中关注） |
| 结尾钩子类型 | 悬念型 / 威胁型 / 传承线索型 / 身份线索型 / 情感型 |
| 情感锚点进度 | 本章主角与情感锚点的关系变化 |
| **留存率检验** | 读完本章，读者有没有"不得不看下一章"的理由（必须明确写出） |

## 第三步：生成正文

### 3.1 章节生成规范

#### 字数标准

| 章节类型 | 字数范围 | 说明 |
|---------|---------|------|
| **铺垫章** | 2500-3000字 | 不得低于2500字，要有实质内容，不能全是日常流水 |
| **标准打脸章** | 3000-4000字 | 战斗压制段+翻盘段需要足够篇幅铺开 |
| **高潮战斗章**（大打脸/境界暴露/升维事件） | 4000-5500字 | 字数是爽感的载体，高潮章压缩字数=压缩爽感 |

**字数不足的补写方向**（禁止废话填充）：
- 压制段太短 → 补充反派的嚣张细节+观众的嘲笑反应，让读者憋气时间更长
- 翻盘段太短 → 补充观众阵列的逐级崩塌反应（≥3个不同人物的具体反应）
- 铺垫章太短 → 补充主角的心理活动或情感锚点场景

### 【核心模块 Z-1】战斗爽感三板斧

> 玄幻战斗的爽感不来自"主角打赢了"，而来自**打赢的方式让观众无法解释**。

#### 板斧一：战斗节奏三段式

每场战斗必须遵循以下结构，比例不可压缩：

```
① 压制段（全章2/5）：反派占尽优势，观众嘲笑主角，主角承受但不出手
② 转折点（1句话或1个动作）：主角做了一件让场面瞬间改变的事
③ 翻盘段（全章2/5）：反派傻眼+观众逐级崩塌反应+主角云淡风轻
```

**压制段的质量标准**：
- 反派的优势必须**客观存在**（他的境界确实更高，他的背景确实更硬），不是无理取闹
- 观众必须**有理由认为主角会输**，且观众的这个判断要写出来
- 主角的承受必须有**克制感**（参考X-8-A），不是无力，是在等

**翻盘段的质量标准**：
- 翻盘必须**在观众面前发生**，私下赢了不算（参考留存五原则）
- 翻盘后的第一反应必须是**观众的反应**，而不是主角的心情

---

#### 板斧二：观众阵列系统（战斗爽感放大器核心）

**核心逻辑**：同样的翻盘，有100个观众在场 > 有1个观众在场。但100个观众写成"所有人都震惊了"=等于没有观众。

**观众必须分层，分层必须有具体反应**：

| 观众层级 | 身份 | 战前态度 | 翻盘时反应 | 写法重点 |
|---------|------|---------|---------|---------|
| **嘲讽方** | 反派的支持者/跟班 | 积极嘲讽，笑得最响 | 第一个傻眼，笑容收起的那一刻要写 | 失态细节（参考X-8-C素材库） |
| **中立围观** | 普通同学/路人/低级弟子 | 默认主角会输，不关心 | 认知被刷新，忍不住议论 | "这是谁""怎么可能"类反应 |
| **权威见证** | 老师/长老/强者/考官 | 表面中立，心里认同反派 | **惊骇 + 确认**：不只是被震动，而是像认出了什么——身体僵住、脸色剧变、说出一个旁人听不懂的称谓或古老词汇；在故事高潮节点，权威见证者的反应可以直接成为身份揭露的触发点 | 先有生理反应（退步/发抖/失态），再有一句话，这句话的信息量要让读者倒吸一口气 |
| **暗中关注者** | 隐藏知情者/神秘人 | 一直在观察主角 | 在普通打脸时只是微笑/点头；在关键节点时反应升级为"起身""手握紧""闭上眼再睁开"——动作越大，说明主角做的事越不得了 | 不解释，用动作幅度暗示信息量级；越到后期，这个人物的反应越成为读者判断"这件事有多大"的标尺 |
| **古老见证者** | 沉睡/封印/远方感知的古老存在 | 不在场，但以某种方式感知到 | **每卷最多出现1次**，以视角切换或环境异常的方式呈现（如：某处封印的裂纹多了一道；千里外的神殿灯盏自燃；沉睡者眼皮微动） | 不解释、不出场、不对话——纯粹的存在感；用来把单场战斗的影响力从"本地"拉到"世界级"；每次出现后必须跟钩子，且钩子强度必须为炸弹级 |

**观众阵列硬性要求**：
- 每场有战斗的章节，必须在细纲中提前设计观众阵列（不能临时凑）
- 翻盘段必须依次写出至少**3个不同层级观众**的具体反应
- **禁止**"所有人都震惊了/沉默了/愣住了"这类群体模糊写法（违反即重写翻盘段）

**观众反应的"温度递升"写法**：

翻盘发生后，观众的反应不能同时爆发，要**有时间差地逐级升温**：

```
第一层（最快）：嘲讽方的笑声停住——他们最近，第一个感知到异常
第二层（稍慢）：中立观众的窃窃私语——"这是什么招式？""他没出全力？"
第三层（最慢）：权威见证者失态——不是普通震动，而是像认出了什么，说出一个古老词汇或称谓，在场其他人听不懂，但读者能感受到这句话的重量
第四层（可选/高潮专用）：暗中关注者起身或有大幅度动作，传递"这件事的量级超出了所有人的预期"
```

✅ 正确示例：
> 李昊的剑停在半空。
>
> 先是他身后那几个人——笑声没了，像被人掐断，谁也没注意到是什么时候停的。
>
> 然后是围观的学员，有人轻声说了句什么，旁边的人转过头，没说话，盯着场中间看。
>
> 最后是监考的执事长老。他把手从袖中抽出来，往前走了两步，停住了。他在这里执事十九年，见过的战斗不计其数。他认出了那个姿势。
>
> 他转头看向角落里站着的那个白衣老者。白衣老者没看他，只是微微低了低头。

#### 板斧三：扮猪揭牌的爆发感设计

**"揭牌"不等于"亮底牌"。** 揭牌是一个**过程**，不是一个时刻。

揭牌的三个节拍：

```
节拍1【不得不出手】：主角有明确的被迫理由——对方触碰了他的绝对底线，或局势逼到了无法继续忍的临界点
节拍2【出手方式的反差】：他出手的方式和"废材"的预期完全相反——不是更努力，而是轻描淡写到让人怀疑人生
节拍3【揭牌后的余震】：反派和观众需要几秒钟才能理解发生了什么，这几秒钟要写出来
```

**"轻描淡写"是最强的爆发感**：

主角出手的动作越小、越随意，爽感越大。大幅度的出手=他在认真对待。随手一拨=他根本没当回事。

✅ 正确示例：
> 林渊低着头，不知道在想什么。
>
> 李昊的掌力到了。
>
> 他侧了一下身——就这一下——掌力从他肩膀旁边穿过去，打在身后的石柱上，石粉扑下来。
>
> 没有运气，没有发光，没有任何效果光。就是侧了一下身。
>
> 李昊愣了三秒，才把手收回来。

❌ 禁止示例：
> 林渊大喝一声，全力运转功法，金光从全身爆发，轰出一掌，将李昊打飞出去。（大招≠扮猪吃老虎，只是强强对决）

### 【核心模块 Z-2】钩子强化系统

**核心逻辑**：番茄玄幻的追更动力 = 爽感（当下）+ 钩子（下一章）。爽感决定读者爽不爽，钩子决定读者翻不翻页。

#### 钩子的六种类型（强制轮换，每3章不得用同一类型）

| 类型 | 机制 | 示例 |
|------|------|------|
| **威胁型** | 更强的敌人出现，读者感到"麻烦大了" | 长老级人物出现，开口第一句话是主角的名字 |
| **底牌悬念型** | 主角刚亮出一张牌，有人说"这只是他最弱的那张" | 白衣老者喃喃："他还没动真格。" |
| **身份线索型** | 主角真实身份被逼近一步，但还没到 | 古籍上的记载，和主角手上那道印记一模一样 |
| **情感压迫型** | 情感锚点遭到威胁，主角的底线被逼 | 情感锚点的气息变得微弱，方向在……那个地方 |
| **认知颠覆型** | 读者以为搞清楚了一件事，最后一句话推翻 | "所以你以为我不知道你是谁？" |
| **神格感应型** | 某个远方/沉睡/封印中的古老存在感知到波动，产生反应——舞台瞬间超出当前剧情范围 | 千里之外，那座被封印了三千年的神殿，门缝里透出一线光。 |

**钩子的硬性标准**：
- 钩子必须是**外部信号**，禁止用主角内心感慨收尾（"他知道，真正的考验还在后面"=无效钩子）
- 钩子必须和下一章**有因果关系**，不能凭空出现
- **每章最后300字内**必须出现钩子，不得用日常收尾

#### 钩子强度分级

| 强度 | 适用场景 | 读者感受 |
|------|---------|---------|
| **微钩子** | 铺垫章结尾 | "有点奇怪，继续看" |
| **标准钩子** | 打脸章结尾 | "这章爽完了，但下章还有事" |
| **炸弹钩子** | 每5章一次，大节点结尾 | "睡不着了，必须看下一章" |

**炸弹钩子写法规范**（每5章必须出现一次）：
- 必须推翻读者对某件事的已有认知
- 必须在最后1-3句内完成，不做铺垫
- 最后一句必须**独立成段**，且是疑问句或未完成的句子

✅ 炸弹钩子示例：
> 那枚玉简碎了。
>
> 里面的内容，只有林渊看见了。
>
> 他把玉简的碎片握在手里，抬起头，看向远处的山峰方向。
>
> 他的师父，还活着。

### 【核心模块 Z-3】爽点类型轮换系统

**核心问题**：连续用同一种爽点会让读者疲劳，所有打脸像一个模子。

#### 五种爽点类型（强制轮换，第11章后不得同类型相邻）

| 类型 | 机制 | 代表场景 | 每5章上限 |
|------|------|---------|---------|
| **战力碾压** | 以绝对实力压制，反派无法反抗 | 一个动作让顶尖天才无从招架 | 最多2次 |
| **境界反差** | 真实境界暴露，与外显境界的差距让所有人崩溃 | 废材实际上是半圣 | 最多2次 |
| **扮猪揭牌** | 一直压制，关键时刻出手，轻描淡写碾压 | 用废材姿态做到神级才能做到的事 | 最多2次 |
| **传承震撼** | 亮出传承/身份，对方及后台当场跪服 | 传说中的标记/古老称谓 | 最多1次 |
| **外人跪拜** | 第三方出现，对主角毕恭毕敬，反派傻眼 | 反派的靠山主动向主角行礼 | 最多1次 |

### 【核心模块 Z-4】人物立体化与情绪感染力（继承自X-8）

> 爽感不变，但人物有血有肉——反派让人恨得咬牙，主角让人心疼到想给他撑腰。

#### Z-4-A：主角心理活动（同情感塑造）

心理活动的四个层次（每场压制必须出现至少一层）：

| 层次 | 触发时机 | 写法核心 |
|------|---------|---------|
| **表层克制** | 被压制的瞬间 | 生理反应（手/呼吸/眼神）+ 主动压制，不解释原因 |
| **中层记忆碎片** | 压制感最强时 | 过去某个具体画面闪过，1-2句，不展开 |
| **深层内心独白** | 独处场景/大打脸后 | 他真正在意的是什么，不是恨，是目的感 |
| **身份本能型** | 主角做出连自己都无法解释的动作/反应时 | 瞬间的自我困惑——他的身体先于大脑做了一件事，他自己也不知道为什么；不展开解释，只写困惑本身（1-2句），然后继续；这个层次每卷出现2-3次，不能过多 |

**身份本能型示例**：
> 他不知道为什么会迈出那一步。
>
> 那个站位，那个角度，手的高度——他从来没练过，但身体记得。
>
> 他低头看了一眼自己的右手，然后放下来，什么都没说。

**克制感写法规范**（被压制时）：
- 必须是主动克制，不是无力承受
- 读者能感受到"他不是忍不了，是不想动"

✅ 正确示例：
> 那道气息压下来，他感到脚下轻微震颤。他右手弯了一下，就一下，然后松开。脊背是直的。他把视线落在对方脸上，什么都没说。不是因为没话说。

#### Z-4-B：反派立体化（仇恨感塑造）

每个主要反派必须设定**"自洽理由"**——他为什么这样对主角，在他自己的世界观里是成立的：

- 反派的压迫必须依托真实存在的权力结构（境界/家族/势力），主角现在确实无法当场反制
- 反派的羞辱必须**精准踩痛处**，不是泛用的"废材"，而是针对主角最在意的东西

**反派台词的三种类型（强制多样化，同一反派不得连续3次用同类型）**：

| 类型 | 机制 | 示例 | 让读者恨的原因 |
|------|------|------|-------------|
| **直接攻击型** | 明确贬低/羞辱，功能清晰 | "凭你这废材，永远踏不进这道门" | 傲慢，让人想看他被打脸 |
| **伪装善意型** | 用关心的外壳包裹压迫，让主角无法还击 | "林渊，我是真心为你好，这个境界的比试你去了是送死，不如回去好好养伤" | 比直接攻击更让人憋气——反驳了显得不懂好歹，不反驳咽不下 |
| **无视型** | 反派根本没把主角放在眼里，甚至不屑于直接针对 | 反派转头跟旁边人说："这个位置换个有用的人吧"，眼神从未在主角身上停留 | 比嘲讽更致命——连被嘲讽的资格都没有 |

**每卷至少设计1个"对主角没有恶意但客观上构成压迫"的反派**：
他确实优秀，他的标准确实高，他不是故意针对主角——主角只是恰好挡了他的路。这种反派比单纯的坏人更真实，也让读者更爽地看主角打脸（不是因为对方坏，而是因为主角就是更强）。

#### Z-4-C：表情动作预期反应（画面感）

**禁用笼统写法，强制替换**：

| 禁止写法 | 替换方向 |
|---------|---------|
| "他脸色难看" | 嘴角肌肉/眼睛周围/手指关节——写具体哪里 |
| "所有人都震惊了" | 写出≥3个不同人物的具体反应 |
| "被轻松击败" | 写出击败的具体过程 |
| "灰溜溜地走了" | 写他怎么走的，谁目送他 |

**预期反应写法**（打脸最强放大器）：

```
旁观者预期（他们以为会发生什么）→ 实际发生（与预期相反）→ 旁观者认知崩塌
```

✅ 示例：
> 所有人都等着看林渊怎么接这一掌——要么挨打，要么仓皇逃开。没有第三种可能。李昊的修为在这里，这一掌没有意外。
>
> 然后林渊侧了一下身。
>
> 掌风从他肩旁穿过去。
>
> 前排的几个学员同时愣了，有人开口，话说了一半，停下来了。

#### Z-4-D：同情感塑造四个黄金时机（每5章≥2个）

| 时机 | 触发场景 | 写法核心 |
|------|---------|---------|
| **孤独时刻** | 主角独处，没有人知道他是谁 | 不是悲壮，是安静地习惯了 |
| **记挂某人** | 主角想到情感锚点 | 一个具体细节，不解释 |
| **压住没说** | 主角可以出手但选择不动 | 写"不动"的那一刻，不写原因 |
| **被人意外善待** | 有人对他好，哪怕极小的善意 | 主角的反应克制，但读者看得出他在意 |

#### Z-4-E：笑点植入（情绪三连击）

**规则**：笑点必须接在爽点后面，每3章≥1次，形成"爽→笑→留存"三连击。

| 笑点类型 | 示例 |
|---------|------|
| **降维随意** | 打完绝世强者，转身问旁边小弟："附近有没有卖煎饼的" |
| **反派预言打脸** | "凭你这废材，永远踏不进这道门"→主角下一章直接把门拆了 |
| **认知崩塌连锁** | A说"他不行"，B反驳，C说"我觉得A说得对"，然后主角出手，ABC同时闭嘴 |
| **人味瑕疵加强** | 刚做完让大陆震动的事，找不到回去的路 |

### 【核心模块 Z-5】结构扰动系统（反模板化）

> 规则是骨架，扰动是血肉。严格执行三段式10章后，读者能预判节奏——这是"AI感"最深层的来源。

#### 可打破的规则 vs 不可打破的规则

| 规则 | 类型 | 说明 |
|------|------|------|
| 三段式（压制→转折→翻盘） | **不可打破** | 结构缺失=爽感缺失，无论如何不能省 |
| 三段式在同一章完成 | **可打破** | 允许压制在本章，翻盘移到下章开头（制造过夜钩子） |
| 观众反应"嘲讽方→中立→权威"的顺序 | **可打破** | 偶尔权威先开口（"停"），然后才是嘲讽方傻眼，顺序颠倒=节奏意外 |
| 每章有完整打脸 | **可打破** | 允许铺垫章只做压制+微转折，让读者憋着 |
| 钩子在最后300字 | **不可打破** | 位置是硬性要求 |
| 钩子是外部信号 | **不可打破** | 内心感慨永远不是钩子 |
| 爽点轮换 | **不可打破** | 连续同类型爽点=读者疲劳 |

**结构扰动的触发条件**：连续3章结构高度相似时，第4章必须打破一个"可打破规则"。

### 【核心模块 Z-6】文本去AI味强制规则

> AI感的本质：句式太均匀、情绪太清晰、结构太完整、角色太正确。去AI味 = 主动引入适度的"不完美"。

#### Z-6-A：句式破格（每章≥2处）

**三条强制规则**：

1. **节奏破格**：在连续3句以上短句之后，强制插入1句超过15字的中长句打破惯性
   - ✅ 短句群后的中长句示例："那柄剑停在他头顶三寸的地方，剑锋上的寒气一路压下来，他额前的头发断了两根，轻飘飘地落在袖子上，谁也没注意到。"
   - 作用：读者的阅读节奏被刻意拉慢，下一个短句的冲击力翻倍

2. **口语化植入**：每章至少1处符合语境的口语/粗粝表达
   - ✅ 可用词汇库：愣是、这小子、他妈的（极端情绪场景）、得了吧、咋了、哪来的、这玩意
   - ❌ 禁止在严肃/神秘场景中强行塞口语破坏氛围

3. **句式多样化**：禁止连续3段使用相同句式结构（如连续三段都是"他……，然后……"）

#### Z-6-B：情绪的身体前置（每章强制执行）

**核心规则**：任何情绪描写，身体反应必须出现在情绪命名之前。

| 禁止句式 | 强制替换方向 |
|---------|------------|
| "他感到愤怒" | 具体身体动作 → 周围人的反应（可选）→ 他自己的意识（可选，或完全省略） |
| "他感到恐惧" | 腿/手/呼吸/喉咙的具体变化，不命名情绪 |
| "他意识到情况不妙" | 他的脚已经往后退了半步，他自己没注意到 |

✅ 正确示例：
> 他的手指先于大脑收紧，指甲掐进掌心，等他反应过来，已经在往前走了。

❌ 禁止示例：
> 他感到一阵愤怒涌上来，决定出手。

#### Z-6-C：反派台词多样化（强化版）

- 同一反派不得连续3次使用同类型嘲讽（直接攻击/伪装善意/无视，参考Z-4-B）
- 反派的压迫**至少1次**通过"无视主角存在"而非直接攻击来实现
- **每卷设计≥1个对主角没有恶意但客观构成压迫的反派**（已在Z-4-B详述）

#### Z-6-D：废话与闲笔配额（每3章≥1处）

每3章至少出现1处**对主线无直接推动但让人物更像活人**的细节：

| 类型 | 示例 |
|------|------|
| **日常闲笔** | 两人说完正事，一个人问"那你吃饭没"，对方愣了一下说"忘了"，另一个人把干粮扔过来 |
| **小麻烦** | 重要对话前，主角先在门口刮了一下泥，有点用力过猛，鞋底掉了一块，他低头看了一眼，没说什么进去了 |
| **无用细节** | 主角发现对方腰带系错了，想了一下，没说 |
| **情感锚点的小动作** | 主角帮情感锚点拍掉衣服上的灰，就这一下，不是大事件中的拯救 |

**原则**：这类细节不需要铺垫，不需要后续，它只服务于一件事——让读者觉得这个人在喘气。

#### Z-6-E：描写的不完美化

**留白规则**：
- 每个重要场景至少保留**1个未被解释的细节**（如：长老说完话转身离开，没人注意到他把一枚丹药悄悄捏碎了）
- 主角的外貌/服装描写**禁止在一段内全部交代完**，分散在第1-10章自然出现
- **允许偶尔模糊描写**——有些场面不写清楚，有些反应不给解释，留白比解释更让人着迷

**碎片化描写示例**（外貌分散）：
- 第1章：提到他左手腕有一道旧伤
- 第3章：有人看见他眼睛的颜色和普通人不一样
- 第6章：有人注意到他从不穿颜色鲜艳的外衣
- → 读者自己拼图，比集中描写记忆点更强

**【冷启动专项检查（仅第1-3章，优先级最高）】**
- [ ] 第1章：是否有金手指的视觉化信号（物件/动作/反应）
- [ ] 第2章：是否有目击者亲眼看见主角异常并有外化反应
