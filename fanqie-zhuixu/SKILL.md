# 赘婿战神极速爽文生成器

**核心公式**：爽感 = 主角被压制的屈辱感 × 翻盘时的身份反差。透支过早 = 后续无牌可打。

## 三线身份透支控制（核心铁律）
| 身份线 | 暴露节奏 | 作用 |
|--------|---------|------|
| 宗师（武力） | 第1卷首批暴露，最表层 | 解决本地冲突 |
| 首富（财力） | 第1卷中段暴露 | 解决商界冲突 |
| 战神（军力） | 第1卷末或第2卷 | 全场最强底牌 |

**严禁前20章同时亮出两条完整身份线。每次暴露必须“暴露一半，留一半”。**

## 输入格式
用户提供主角名、岳父家背景、故事起点。若无则默认：林峰，27岁，退役三年，入赘苏家，从婚礼当天开始。
收到输入后直接执行第一步，无需额外确认。

## 主角类型选择器（前置）
生成前先判定类型，影响爽点节奏。默认类型C（能力压制）。
- **A.废材逆袭**：从弱到强。爽点来自成长，触底事件后觉醒。
- **B.身份反转**：神王归来。爽点来自一层层揭开的反差。
- **C.能力压制**：能力已有，被身份/环境遮盖。**爽点是“被重新看见”**，不是变强。主角出手必须是**外部事件触发**（路人受伤/突发状况/被迫卷入），出手本身是主动的，但**出手的契机是被动的**。结果必须超常识，主角本人轻描淡写。禁止主角内心独白“知道很强”，禁止主动解释能力来源。前10章爽点类型以“被动专业展示”为主，不得与“身份威慑”相邻。

**类型C专用铁律**：克制必须是主动压制（身体的微小停顿后被自己按下去），而非被动忍受。读者必须感受到“他不是忍不了，是不想动”。

## 第一步：人设与鄙视链架构（≤1500字）
### A. 主角人设卡
必须含：表面身份（废物赘婿）、三线具体设定（门派/集团/军衔）、核心性格（隐忍表+霸道底）、口头禅、每线的被迫亮牌条件。

### B. 鄙视链（6级，前20章只处理L1-L2）
L1岳父家内部→L2本地豪门→L3省级财阀→L4全国顶级家族→L5军政圈→L6国际势力。

### B-plus. 反派类型
若选类型C，主要反派为“阶级蔑视型”：羞辱来自无视、归类、轻描淡写的定性，而非直接辱骂。打脸方式必须是反派认知崩塌，通过一个**读者可见的具体行为**外化：收起离婚协议/第一次叫“先生”/放下酒杯没再喝/走到门口停了一步。**禁止反派只在内心震动而外表无变化，不要求下跪道歉。**

### C. 岳父家人物表
岳父、岳母、小舅子（第3章第一个打脸对象）、妻子。标注鄙视程度和打脸时机。

**自检**：三线触发条件是否独立？鄙视链是否无跳级？妻子有无成长线？反派失败方式是否有记忆点？

## 第二步：前20章细纲（留存率引擎）
### 关卡一：第1-3章 破冷启动（铁律六条，违反即重写）
①第1章第一句话必须是冲突或羞辱，首段150字内出现羞辱场景。
②第1章必须有金手指**视觉化暗示**（物件/符号/动作），不许纯背景。
③第2章必须有一个非主角的**目击者亲眼看见能力展示**并做出外化反应（不能只是“震惊”，必须有具体行为）。
④第3章第一个打脸必须**≥3人公开目击**，且至少写出1个具体反应。
⑤前3章主角必须有≥1次**主动选择介入**（他自己看见，走过去，做了），不能所有行动起点都是“被叫去/被请求”。
⑥第3章打脸段落翻盘后必须“炸”：必须出现≥2个“热信号”——短句独立成段（≤5字/句，连续3句）、某人的具体身体反应（手抖/后退/屏息）、环境反差（刚才喧闹现在只剩仪器声）、反派/见证者只说了半句话。**禁止“走廊安静下来”“主角云淡风轻”“妻子若有所思”等冷收尾。** 结尾必须有“更大麻烦正在来”的外部信号（神秘人物出现/手机亮屏/陌生来电等）。

### 关卡二：第4-10章 建立爽感惯例
节奏：每2章一爽点。第4章经济羞辱，第5章打脸岳母（宗师暗示），第7章打脸小富豪（宗师半暴露），第8章打脸岳父（宗师完整），第10章第一个大打脸（宗师全开+财力暗示）。**检验：第10章读者不能知道战神身份。**

### 关卡三：第11-20章 升级钩子
第11-13章财力线完整暴露，第14-16章引入L3反派+追查身份，第17-19章打脸L3+战神线第一条线索，第20章战神身份被半猜但主角否认。

每章细纲必须含：爽点类型（七选一）、爽点等级、身份暴露等级、反派行动、主角反制方式（禁止模糊描述如“轻松击败”）、结尾钩子类型、妻子情感进度、留存率检验（具体理由，不能只写“读者会想看”）。

**铺垫章不得超过总章数1/3。**

## 增强模块（核心规则集成）

### X-1：爽感类型轮换（七种，第11章起强制轮换）
| 类型 | 上限 | 关键要求 |
|------|------|---------|
| 武力震慑 | 每5章≤2次 | — |
| 财力碾压 | 每5章≤2次 | — |
| 智谋设局 | 每5章≤1次 | 主角提前布局，反派自己走入陷阱 |
| 身份威慑 | 每5章≤1次 | 亮身份，对方腿软 |
| 外人跪拜 | 每5章≤1次 | 第三方对主角毕恭毕敬 |
| **被动专业展示** | 每5章≤2次 | **主角被动触发，结果超常识，本人轻描淡写。不得与身份威慑相邻。** |
| 指令暗线 | 每5章≤2次 | 主角幕后安排，结果呈现时主角不在场。**不得连续使用。** |

**规则**：同一类型不得连续；武力+财力不得相邻；每10章内必须出现一次“智谋设局”或“外人跪拜”。第1-10章不强制轮换，以武力为主。

### X-2：主角人味瑕疵（可选，强烈推荐）
给主角加一个与战力/财力无关的小毛病（路痴、某种食物执念等），第2-4章首次出现，每5章至少复现一次。大打脸后立即接瑕疵场景，形成“爽感→亲近感”切换。

### X-3：反派失败方式强制视觉化（防模糊处理）
**每章打脸前，先输出前置锁定**：
“本章反派失败方式预锁定：反派[姓名]，失败触发点[一句]，视觉化场景[含具体肢体/物件细节]，围观者反应[谁+动作]，反派最后一句话/动作[非灰溜溜]。”

**失败类型素材库**（每5章内每类≤1次）：
- 身体失控（跪地、颤抖、撞翻椅子）
- 物件损毁（表碎、合同撕毁、支票退回）
- 被自己人背刺（手下先跪、靠山来电道歉）
- 当众出丑（被保安请出、麦克风传遍求饶话）

**黑名单**（出现即重写）：灰溜溜走了、脸色难看离开、被轻松击败、认输了、所有人都震惊了。

### X-4：情绪温度强制切换
打脸场景必须“铺垫段冷→转折句（一句话独立成段）→翻盘段热”。翻盘段热度必须来自围观者反应，主角越淡围观者越失控。**禁止写主角热血沸腾/嘴角扬起/内心激动**——主角的反应只能是“把针擦干净/推回本子/站起来走了”，热度全部交给围观者。必须出现≥2个“热信号”（短句独立成段、具体身体反应、环境反差）。

**前三章情绪分配**：第3章翻盘段必须“炸”（热信号≥2个）。

### X-5：番茄留存十二条铁律（精简版）
1. 爽点必须被曾羞辱主角的人在场或间接感知。
2. 前5章内必须≥1次主动选择介入，之后每5章至少1次。
3. 羞辱者必须在同章或下一章内遭遇由主角触发的、外化可见的反噬，并有旁人在场反应（旁人必须有一个可见动作，不能只写“气氛变了”）。
4. 外部悬念人物出现后，下一章内必须让局面产生压力，同一人物不得连续3章只观察不接触。
5. 每章内必须有≥1次情绪节拍从憋屈到爽/燃的切换（含微切换，如主角在本子上记了一笔让读者感到“他在等待”）。
6. 每章必须有至少一个可传播的台词或画面（冷刀子型：伏笔、信息密度、反差、精准打击）。
7. 见证者对主角的认知深度必须逐次递增（一级：觉得不对劲→二级：认出专业→三级：认出身份）。
8. 有围观者时必须安排“嘴替”，公开场合直接开口，家庭场合可间接嘴替（沉默/转移话题），但嘴替的话必须指向羞辱者，且有羞辱者的可见反应。
9. 每章必须有处境发展触点，且第一章主矛盾（如秦家鄙视）每章至少出现一次触点（一句话/一个物件/一个人的态度变化）。
10. 羞辱者的态度转变必须经历至少2个有场景支撑的步骤，禁止一步跳转。
11. 有姓名的被帮助者本卷内必须有一次回响（感谢/名声/资源）。
12. 配角内心状态必须通过行为动作传达，禁止内心独白段落直接说明。

### X-6：场景执行节拍模板
**模板A：公开羞辱反噬（四节拍）**
1. **羞辱落地**：羞辱者具体动作+旁人反应；主角“留痕动作”（记账/看一眼/动了一下手指）。禁止主角无任何留痕。
2. **能力展示**：被动触发，精准描写（部位+角度+力道），结果用数字/状态量化。禁止主角主动说“我来”。
3. **见证者认知升级**：见证者具体行为（不能只写“愣住”），认知深度选一级/二级/三级。禁止连续两章用同级。
4. **嘴替炸裂**：羞辱方内部人补刀，羞辱者被迫当众做具体动作（收起笑/放下东西/说话），主角不开口。禁止主角本人当嘴替。

**模板B：日常铺垫章（三节拍）**
1. **主矛盾触点**：秦家人给压力/悬念信号/能力暗示，主角留痕。禁止全章无秦家人出场。
2. **微爽点**：对方轻视的同时主角已悄悄占优（数字/物件量化），对方当时不知。必须有旁人见证或反应，禁止纯内心独白。
3. **钩子埋设**：恐惧/身份/关系型钩子，在最后1/4篇幅出现。禁止无任何悬念信号。

**模板C：沉默型克制反击（三节拍）**
1. 对方出招，主角用行动/数字/一句话接招并留痕。
2. 行动结果用可测量单位呈现（克数/时间/金额），禁止只有形容词。
3. 延迟反噬：主角不在场时，对方独处时反应暴露（放下杯子/收起笑），禁止主角在场目睹。

### X-7：高频套语禁用清单
每章扫描： “停了一下”≤2次，“有什么东西”≤1次，“把X放进去/收下”≤1次，“没有说话”不能裸用连续出现，“低下头+抬起头”组合≤2次。超限强制替换为行为描写或换行短句。

### X-8：人物立体化与情绪感染力（精简）
- **主角心理**：被羞辱时用主动压制的生理细节（手停在原位，脊背挺直，然后才移开视线），而非被动忍受（愣住/难受）。独处时深层独白只写目的感和重量感，不写恨。
- **反派立体**：每个主要反派必须有自洽理由，羞辱必须精准踩痛处（针对软肋），依托真实权力结构让主角无法当场反击。
- **表情动作**：禁用“脸色难看/很震惊/愤怒地看着/感动流泪”等笼统写法，替换为具体部位（眼角/手指/喉结/呼吸等）。每章重要人物须有≥2个具体动作。
- **预期反应**：打脸前必须写旁观者预期（他们以为会怎样），然后被现实打破。
- **笑点植入**：每3章必须出现1次笑点（降维凡尔赛/反派预言打脸/妻子认知崩塌/人味瑕疵），**必须接在爽点后，禁止出现在铺垫段和钩子前**。

### X-10：最小单位互动（情感锚点）
情感推进用极小互动完成：让路、一条消息未读、门缝的光、说了“谢谢”然后关门。第一卷每3章至少1次，每次≤6行。主角被动接受，不解释感受，用行为替代内心独白。禁止台词直接表达情感，禁止双方确认感情。

### X-11：擦边叙事（按需使用）
**核心技术**：写感官细节（气味/温度/触感＞视觉），到达临界点前切走。临界点判断标准：当描写已让读者心跳加速、即将脑补画面时，用一个外部打断（电话/敲门/外面声音）或主角的主动终止动作（转过身/退后/松手）切换场景。切点必须是行为或外部事件，不能是内心独白“他停了下来”。主角始终被动或漠然，不主动发起。
**密度**：铺垫章低烈度（A类日常接触），打脸章中烈度（与情感推进联动），高潮章可1-2次。**禁止前3章用中高烈度场景，禁止连续2章用D类情绪爆发擦边。**
联动：X-10的最小互动可升级为A类擦边（加一个温度/气味细节）。

## 第三步：生成正文
**字数标准**：铺垫章≤2200字，打脸章2800-3000字，高潮章4000-5000字。字数不足时补压制段/翻盘细节/围观者反应，禁止废话。
**打脸三段式**：压制段（约1/3，让读者憋气）→转折（1句话独立成段，切断前文节奏）→翻盘段（约1/3，围观者反应炸裂）。比例为感觉比例，非精确字数切割，但翻盘段不得短于压制段的一半。
**章节结构**：前1/3建立冲突，中1/3能力展示或主动行动，后1/3反噬+嘴替，结尾钩子。
**铺垫章升级为双层章**：表层日常压制，底层必须埋入能力伏笔（主角做了超出认知范围的事）和情感锚点（关系松动信号）。禁止纯铺垫章。

**妻子情感线节奏**：第1-5章失望，第6-10章困惑，第11-15章心动，第16-20章依赖。每5章推进一次，必须由具体事件触发，禁止“因为他厉害”。

**结尾铁律**：必须是外部事件钩子（手机亮/陌生来电/有人出现），禁止内心感慨总结。同一意象（走廊/壁灯/影子等）不得连续出现超过3章。

## 每章生成后终极自检
**冷启动专项（第1-3章）**：
- [ ] 第1章首段冲突+金手指视觉暗示
- [ ] 第2章目击者看到并反应（具体行为）
- [ ] 第3章打脸≥3人目击，翻盘段“炸”且有热信号≥2个，结尾外部信号
- [ ] 前3章主角主动出手≥1次（主动选择介入，非被叫）

**常规检查（全章）**：
- [ ] 字数符合章节类型
- [ ] 完成压制→转折→翻盘三段式，翻盘段未短于压制段一半
- [ ] 翻盘段情绪温度明显切换，热度来自围观者非主角，有≥2个热信号
- [ ] 爽点类型轮换合规（第11章起）
- [ ] 反派失败方式已前置锁定且无黑名单词汇，类型未重复
- [ ] 身份暴露程度与细纲一致
- [ ] 结尾钩子与细纲一致，且为外部事件，非内心感慨
- [ ] 妻子情感线有推进
- [ ] 人味瑕疵（若启用）本章或近5章内出现
- [ ] 铺垫章→双层章结构，含能力伏笔+情感锚点
- [ ] 本章有能力展示时：有曾羞辱者感知
- [ ] 主动出手频率合格
- [ ] 羞辱者反噬完成（同章/下章，有外化行为+旁人反应）
- [ ] 外部悬念人物若出现，未超3章不接触
- [ ] 情绪节拍切换≥1次
- [ ] 处境有发展触点，第一章主矛盾有触点
- [ ] 羞辱者转变有步骤，未跳跃
- [ ] 有姓名被帮助者若已过本卷需有回响（非本章必须）
- [ ] 配角无内心独白
- [ ] 有可传播台词/画面
- [ ] 见证者认知递增
- [ ] 有围观者时安排了嘴替，且羞辱者有反应
- [ ] 套语未超限
